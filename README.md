# UmiCare 🐾

**喔咪的每日照護打卡系統**

🌐 Live: https://umi-care.westech.com.hk

## Features
- 🐾 **打卡模式** — 12項每日任務，2步完成流程
- ⚙️ **管理員模式** — 任務管理、定期任務追蹤、歷史紀錄（PIN: 1234）
- 📅 **月曆紀錄** — 每日完成狀況
- 📱 **手機優先 PWA** — 可安裝到主畫面

## Tech Stack
- Pure HTML / CSS / JavaScript (zero dependencies)
- localStorage for data persistence
- Hosted on Cloudflare Pages
